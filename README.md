# Student Management System

## 📌 Overview

This is a Java-based console application designed to manage student records efficiently. The system allows users to perform basic operations such as adding, viewing, searching, updating, and deleting student data.

---

## 🚀 Features

- Add new student records
- View all students
- Search student by ID
- Update student details
- Delete student
- Save student data to a file
- Exception handling for invalid inputs

---

## 🛠️ Technologies Used

- Java (Core Java)
- OOP Concepts (Encapsulation, Constructor, Classes & Objects)
- Java Collections (ArrayList)
- File Handling (FileWriter)
- Exception Handling (try-catch)

## ▶️ How to Run

1. Clone the repository:

   ```
   git clone <your-repo-link>
   ```

2. Navigate to the project folder:

   ```
   cd StudentManagementSystem
   ```

3. Compile the Java files:

   ```
   javac src/*.java
   ```

4. Run the program:

   ```
   java -cp src Main
   ```

---

## 💡 Usage

- Follow the on-screen menu
- Enter choices to perform operations
- Use "Save to File" option to store data permanently

---

## ⚠️ Notes

- Input validation is handled using exception handling
- Marks should be between 0 and 100
- Data is stored temporarily unless saved to file
